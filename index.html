WinRAR Clone

Uma solução poderosa para compactação e descompactação de arquivos

O que é o WinRAR Clone?

O WinRAR Clone é um programa desenvolvido em Python utilizando a biblioteca PyQt5 para fornecer uma interface gráfica amigável. Ele permite aos usuários compactar e descompactar arquivos e diretórios com facilidade e eficiência. Além disso, o programa oferece uma opção para visualizar detalhes dos arquivos e diretórios selecionados antes da compactação.

Funcionalidades

  • Interface Gráfica Intuitiva: Uma interface limpa e fácil de usar, desenvolvida com PyQt5.
  • Compactação de Arquivos e Diretórios: Selecione múltiplos arquivos ou diretórios para compactar em um único arquivo ZIP.
  • Descompactação de Arquivos ZIP: Extraia o conteúdo de arquivos ZIP para um diretório especificado.
  • Visualização de Conteúdo: Visualize os detalhes dos arquivos e diretórios antes de compactar.
  • Menu de Contato: Acesse facilmente os detalhes de contato para suporte e mais informações.
